Gaussian elimination.

This requires SquareMatrix, so you have to copy-paste the code for it as well if you want to use this for online judges.

ElementaryOp is an enum of elementary row operations:

  • Swap(i, j) swaps the i-th and j-th row.
  • Mul(i, k) multiplies the i-th row by k.
  • Add(fr, k, to) adds k times the fr-th row to the to-th row.

The following methods are added to SquareMatrix:

  • apply_row_op(op) applies the given elementary operation in $O(n)$.
  • make_ref() turns the matrix into a row-echelon form, and returns the determinant in $O(n^3)$.

Caution

apply_row_op panics if the operation is invalid:

  • Swap(i, j) where i == j.
  • Mul(i, k) where k == T::default().
  • Add(fr, k, to) where fr == to.
