Module ps_snippets::graph::degree_ordered_graph

source ·
Expand description

A graph DS for properties of sum(min(d(u), d(v))) over edges uv.

The value of that expression is $O(E \sqrt E)$, making it possible to e.g. enumerate all triangles in a simple graph.
